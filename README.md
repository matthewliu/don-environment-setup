# Dev Environment Setup

Setup guides for provisioning new development machines for coding with Claude Code, Cursor, and the claude-toolkit.

## Guides

- **[windows-setup.md](windows-setup.md)** -- Windows 11 Home (WSL2-first approach). Tiered priority: get coding in one sitting, add tools as needed.
- **[mac-setup.md](mac-setup.md)** -- macOS Apple Silicon. Mirrors an existing environment. Includes one-shot install script and dotfile migration checklist.

## What's Covered

- Languages: Node.js, Python, Rust (optional)
- Databases: PostgreSQL 16, Redis, SQLite
- AI tools: Claude Code, Codex CLI
- IDE: Cursor
- Deployment: Railway, Vercel
- Claude Toolkit: MCP review server, slash commands, agents
- Cloud accounts: what to sign up for and when
- Python virtual environments (Windows-specific tutorial)
